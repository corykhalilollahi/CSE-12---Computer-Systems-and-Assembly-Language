Lab 1: Intro to Logic Simulation

-----------------------------
Description

In this lab, 4 input switches in MML are used to affect three different output sections by using distinct logic diagrams and methods.

-----------------------------
Files

Lab1.lgi

This file opens up the MML program that the logic circuits were created in.

-----------------------------
Instructions

This lab is run on MML. Click the green "Run Simulation" button to begin the simulation. Turn the four switches On/Off in different combinations and observe how the
three output sections change based on each input.
