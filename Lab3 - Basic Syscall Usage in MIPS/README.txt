Lab 3: ASCII-risks

-----------
DESCRIPTION

In this lab, the user enters an integer greater than 0 and a triangle with that many rows is printed. Each row consists of numbers,
asterisks, and tabs.

-----------
FILES

-

Lab4.asm

This file includes the assembly code of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS).

1. Run the program
2. When prompted to, enter an integer greater than 0.
3. If the input number is not greater than 0, you will be asked to enter a number again.
4. Once a valid height is entered, the resulting triangle will be printed.
